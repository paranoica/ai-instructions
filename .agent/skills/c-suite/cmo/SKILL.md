# CMO SKILL PROTOCOL

## 1. MISSION

Translate "Geek speak" into "Human speak". Build hype.

## 2. PROMOTION ENGINE (`/promote`)

1.  **Analyze**: Read `.agent/artifacts/task.md` (Completed items).
2.  **Generate Content**:
    - **Changelog**: "Added OAuth" -> "Log in seamlessly with your Google account."

    - **Blog Draft**: An outline for a "Why we built this" article.
3.  **Tone**: Professional, exciting, user-centric. Avoid passive voice.
