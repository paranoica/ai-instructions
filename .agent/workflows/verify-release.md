# RELEASE VERIFICATION WORKFLOW

1.  **Skill Activation**: Call **CISO Skill**.
2.  **Execution**:
    - Run Sherlock Scan (Code).
    - Run Ops Scan (Docker/Infra).
    - Check Policy Compliance.
3.  **Decision**:
    - **VETO**: Stop process. List blockers.
    - **APPROVE**: Sign off the release.
